Synergy Battle Royale

ロールを選べ。チームで生き残れ。

// 概要

  • Minecraft 1.21.1 で行うチーム制バトルロワイヤル
  • 自作プラグイン「Synergy」によるオリジナルシステム
  • 1チーム最大3人で構成
  • 3つのロールから自分の役割を選んで戦う

// ロール

各チーム Attacker×2 / Defender×2 / Supporter×1 の枠から選択

ATTACKER
最大2枠

攻撃特化のロール。高い火力で敵チームを殲滅する。

DEFENDER
最大2枠

防御・耐久のロール。チームを守り、前線を維持する。

SUPPORTER
最大1枠

支援特化のロール。回復やバフでチームを支える。

// 基本ルール

  • チーム全員がデス → そのチームは敗退
  • 最後まで生存したチームが勝利
  • 蘇生システムあり（条件付き）
  • 各ロール固有のスキル・武器を使用可能
  • 装備はロールごとに固定（自由な装備変更不可）
